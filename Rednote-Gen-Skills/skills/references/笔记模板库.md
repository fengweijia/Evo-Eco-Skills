# 小红书笔记模板库

## 类型1：干货教程类

### 结构
```
[开头Hook]
一句话戳痛点/悬念/成果

[自我介绍]
我是谁+为什么有资格说

[核心内容]
- 步骤1/要点1
- 步骤2/要点2
- 步骤3/要点3
（每个步骤都有具体例子）

[总结]
简单回顾+价值强调

[互动引导]
提问/投票/评论区见

[Tags]
#主题 #相关标签1 #相关标签2
```

### 示例
```
姐妹们！终于有人把职场报销讲清楚了！

我是XX，在大厂做财务5年...

1️⃣ 报销单怎么填
  - 日期、项目、金额...
2️⃣ 发票怎么贴
  - 鱼鳞贴法...
3️⃣ 常见被驳回原因
  - 这几点千万别错...

记住这3点，报销一次过！

你们遇到过什么报销问题？评论区告诉我～

#职场#财务#报销#干货#职场技能
```
