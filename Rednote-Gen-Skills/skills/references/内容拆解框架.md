# 爆款内容拆解框架

## 1. 内容结构分析

### 开头钩子（3秒法则）
- **痛点开场**：一句话戳中目标受众的焦虑/需求
  - 例："工资赚得少？是因为你不懂这个"
- **悬念开场**：引发好奇，让人想继续看
  - 例："我靠这个方法3个月涨粉10万"
- **成果开场**：先展示结果，再解释原因
  - 例："从0到1万粉，我做对了这3件事"

### 中间内容
- **信息密度**：每个视频/笔记传达1-3个核心观点
- **案例支撑**：用真实案例/故事来论证观点
- **干货输出**：提供可操作的建议或方法论
- **情绪价值**：让读者有共鸣、有收获感

### 结尾引导
- **互动引导**：提问、投票、评论区见
- **价值引导**：收藏、分享、关注
- **行动引导**：点击链接、预约直播

---

## 2. 选题特征分析

### 话题类型
| 类型 | 特点 | 案例 |
|------|------|------|
| 痛点型 | 解决受众的实际问题 | "不会报销？教你3招" |
| 热点型 | 蹭当下热门话题 | "XX事件爆火因为..." |
| 干货型 | 提供实用知识技能 | "5个职场沟通技巧" |
| 情感型 | 引发情感共鸣 | "30岁职场危机" |
| 种草型 | 推荐产品或服务 | "这个APP太好用了" |

### 标题结构
- **数字+悬念**："5个方法让你的..."
- **痛点+方案**："工资低？因为你不懂..."
- **对比+反差**："从前vs现在"
- **身份+场景**："职场人必看的..."
- **情绪+感叹**："太离谱了！"

### 关键词布局
- 核心关键词（标题必须包含）
- 长尾关键词（正文自然融入）
- 热点关键词（适当蹭热点）
- 品牌关键词（建立认知）

---

## 3. 形式特征分析

### 视频内容
- **时长**：小红书视频1-3分钟最佳
- **节奏**：每15秒一个爆点
- **字幕**：关键信息加字幕
- **BGM**：选择热门背景音乐

### 图文内容
- **篇幅**：300-500字最佳
- **图片**：6-9张图，封面最关键
- **排版**：分段清晰，关键信息突出
- **标签**：3-8个相关标签

---

## 4. 数据表现分析

### 核心指标
- **点赞率**：内容质量指标（1-5%优秀）
- **收藏率**：实用价值指标（2-10%优秀）
- **评论率**：互动深度指标（0.5-2%优秀）
- **转发率**：传播能力指标（0.3-1%优秀）
- **涨粉转化**：关注转化指标

### 爆款特征
- 点赞 > 1000 为小爆款
- 点赞 > 10000 为中爆款
- 点赞 > 100000 为大爆款

## 5. 可复用公式

### 公式1：痛点+方案+案例
```
[开头：戳痛点] 90%的职场人都...
[中间：给方案] 3个解决方法是...
[结尾：引导] 觉得有用就收藏
```

### 公式2：结果+过程+悬念
```
[开头：秀成果] 我如何1个月涨粉5万
[中间：讲故事] 过程是这样的...
[结尾：留悬念] 下期讲...
```

### 公式3：对比+原因+建议
```
[开头：对比] 同样是工作，为什么他...
[中间：分析] 原因在于...
[结尾：建议] 建议你...
```
