## 🔧 功能模块设计

### 模块1: 账号定位助手
**功能**: 接收和管理用户输入的账号定位信息

**输入字段**:
- 账号名称
- 运营目标（单选/多选）
  - 涨粉变现
  - 品牌曝光
  - 知识分享
  - 种草带货
- 内容方向（标签选择+自定义）
- 目标受众描述
- 差异化定位关键词

**输出**:
- 结构化的账号画像
- 定位分析报告
- 后续处理任务ID

---

### 模块2: 对标账号搜集器
**功能**: 在短视频平台（抖音/快手/B站）搜集同领域对标账号

**能力**:
- 搜索指定内容方向的热门账号
- 提取账号基本信息（粉丝数、获赞数、内容频次）
- 识别账号类型（专家型/泛娱乐型/种草型）
- 排序算法：按相关性/热度/更新频率

**输出**:
- 对标账号列表（TOP 10-20）
- 账号分析卡片（每账号包含：简介、爆款率、更新频率）

---

### 模块3: 内容拆解引擎
**功能**: 对对标账号的爆款内容进行结构和规律拆解

**拆解维度**:
1. **内容结构**
   - 开头钩子（3秒法则）
   - 信息密度分布
   - 结尾引导方式
2. **选题特征**
   - 话题类型（痛点/热点/干货）
   - 标题结构
   - 关键词布局
3. **形式特征**
   - 视频时长/图文篇幅
   - 视觉风格
   - 互动设计
4. **数据表现**
   - 点赞/评论/收藏/转发比例
   - 爆款率统计

**输出**:
- 对标账号内容分析报告
- 可复用的爆款公式
- 内容模板库

---

### 模块4: 主题规划器
**功能**: 结合账号定位和热点，自动规划制作主题

**输入**:
- 账号定位画像
- 对标账号分析结果
- 最近7天平台热点

**输出**:
- 3-5个推荐制作主题
- 每个主题包含：
  - 主题名称
  - 内容形式（图/文/视频）
  - 预期效果
  - 参考案例
- 用户确认接口

---

### 模块5: 笔记制作引擎
**功能**: 用户确认主题后，自动完成笔记全要素制作

**产出要素**:
- [ ] **标题**: 至少3个备选（刺痛点/数字感/悬念式）
- [ ] **正文**: 结构化内容（开头发汗+中间干货+结尾引导）
- [ ] **标签**: 3-8个相关标签
- [ ] **封面图**: AI生成或设计建议
- [ ] **发布时间建议**: 根据受众活跃时间

**内容类型支持**:
- 图文笔记
- 视频笔记
- hybrid混合模式

---

### 模块6: 数据监控与优化
**功能**: 持续追踪已发布笔记数据，给出优化建议

**监控指标**:
- 阅读量、点赞、收藏、评论、分享
- 完播率/阅读完成率
- 涨粉转化

**优化建议**:
- 标题A/B测试建议
- 发布时间优化
- 内容迭代方向

## 📊 技术架构

### 多Agent架构
```
                    ┌─────────────────┐
                    │   协调Agent     │
                    │ (Orchestrator)  │
                    └────────┬────────┘
                             │
        ┌────────────────────┼────────────────────┐
        │                    │                    │
        ▼                    ▼                    ▼
┌───────────────┐  ┌───────────────┐  ┌───────────────┐
│ 账号定位Agent │  │ 对标分析Agent │  │ 创作生成Agent │
│ (Profiler)    │  │ (Analyzer)    │  │ (Generator)   │
└───────┬───────┘  └───────┬───────┘  └───────┬───────┘
        │                  │                  │
        └──────────────────┼──────────────────┘
                           ▼
                    ┌───────────────┐
                    │  优化反馈Agent │
                    │ (Optimizer)   │
                    └───────────────┘
```

### 数据流
1. 用户输入 → 存储到用户画像数据库
2. 对标搜索 → 外部API调用（抖音/快手搜索）
3. 内容拆解 → LLM分析 + 模板匹配
4. 主题规划 → LLM生成 + 热点接入
5. 笔记制作 → LLM生成 + 可选AI绘图
6. 数据监控 → 模拟或接入小红书数据接口
