## 2️⃣ AI绘图（封面图生成）

### 推荐方案

#### 方案A: Stable Diffusion API
```python
# 本地部署 Stable Diffusion
# 项目: AUTOMATIC1111/stable-diffusion-webui
# 启动API: python webui.py --api

import requests

def generate_cover(prompt, style="minimalist"):
    response = requests.post(
        "http://localhost:7860/sdapi/v1/txt2img",
        json={
            "prompt": f"{prompt}, {style} style, social media cover",
            "negative_prompt": "low quality, blurry",
            "steps": 20,
            "width": 1242,
            "height": 1660  # 小红书封面比例
        }
    )
    return response.json()["images"][0]
```

#### 方案B: OpenAI DALL-E
```python
import openai

def generate_cover(prompt):
    response = openai.Image.create(
        prompt=f"{prompt}, social media cover image, minimalist style",
        n=3,
        size="1024x1024"
    )
    return [img.url for img in response.data]
```

#### 方案C: 阿里云通义万相（国内可用）
```python
import dashscope
from dashscope import Generation

def generate_cover(prompt):
    response = Generation.call(
        model='wanx-v1',
        prompt=prompt
    )
    return response.output.results[0].url
```
