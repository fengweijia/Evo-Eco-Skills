---

## 📋 项目概览

**核心理念**: 用AI帮助用户自主运营小红书账号，从定位→对标→拆解→创作→优化全流程自动化。

**产品形态**: 多Agent工作流的AI Skills（可被Claude Code、OpenClaw调用）

**技能触发词**: 小红书、账号运营、种草笔记、对标账号、内容创作

---

## 🎯 核心功能（6大模块）

1. **账号定位输入** - 名称、目标、内容方向、受众画像
2. **对标账号搜集** - 自动搜索同领域对标账号（模拟数据）
3. **内容拆解分析** - 学习爆款规律，提取可复用公式
4. **主题规划** - 结合定位+热点，规划3-5个制作主题
5. **笔记制作** - 自动生成标题、正文、封面、标签
6. **数据监控优化** - 追踪数据，持续迭代优化
